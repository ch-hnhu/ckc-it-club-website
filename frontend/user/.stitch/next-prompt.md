A dedicated Events page for CKC IT Club listing all upcoming and past events — hackathons, workshops, seminars.

**DESIGN SYSTEM (REQUIRED):**
This is a Soft Neobrutalism page for CKC IT CLUB, a Vietnamese student IT club.

Visual Style:
- White (#FFFFFF) background with generous whitespace
- Cards with: 2px solid black border, 12-16px border-radius, 4px 4px 0px black box-shadow, pastel backgrounds
- Primary CTA button: #A3E635 lime-green background, black border, black text, bold font
- Secondary button: white background, black border, black text
- Font: Be Vietnam Pro for headings (bold), Inter for body text
- Hover: scale(1.03), deeper shadow
- Layout: clean, playful, Gen Z friendly, well-spaced
Color accents for cards: #D2FAE5, #BFD9FE, #FFDEDE, #FEF3C8, #FAE9FF, #F5D1FE
Brand primary: #A3E635 (lime green)
DO NOT USE: dark navy, glowing blue effects, glassmorphism, cyberpunk aesthetics.

**Page Structure:**
1. Header with back navigation to landing page
2. Hero: "Sự kiện & Hoạt động" with subtitle and upcoming event count badge
3. Filter bar: All | Hackathon | Workshop | Seminar | Meetup (horizontal pill filter)
4. Events grid — 3 columns:
   - Each card: date badge (day + month), title, tag, location, time, short desc, "Đăng ký" button
   - Upcoming events: pastel accent background
   - Past events: gray/muted style
5. "Tổ chức sự kiện" CTA section at bottom
6. Footer
